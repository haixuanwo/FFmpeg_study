# google test

单元测试使用谷歌的单元测试模块
use google's unit test module

git clone https://github.com/google/googletest.git
cd googletest
mkdir build
cd build
cmake ..
make
make install

# 编译单个测试文件
例：make one SRC=src/01_decode.cpp
